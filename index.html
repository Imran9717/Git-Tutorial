GIT TUTORIAL

S.no Type Command Use
1 Config git config --global user.name "username" This command is use for setting the user configuration.
git config --global user.email "userEmail"
2 Initialize git init It is used for initializing or creating git repository.
3 Add git add filename This command is used for adding single file to the git repository.
git add . It is used for adding all file to the git repository.
4 commit git commit -m "message" This command is used for saving the changes of file with message.
5 File status git status This command is shows all files status.
git status -s It help to see files status without extra detials.
6 Commit Detail git log Use to show all commit details.
7 Switch git checkout [commit id] This command is used to see and use previous commited file.
git switch - / git checkout master Move to previous or master commit.
